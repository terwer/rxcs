## 截图规则

1. 必须在目标页面稳定显示时截图
2. 只裁目标控件本体，不要带太多背景
3. 不要带鼠标、手指、高亮框、辅助线
4. 保持原始清晰度，不要二次压缩
5. 文件名必须和下面要求完全一致
