## ADDED Requirements

### Requirement: 选区页就绪检测
系统 SHALL 在执行选区动作前确认当前页面已经进入选区页。

#### Scenario: 选区页特征存在
- **WHEN** 系统准备执行选区流程
- **THEN** 系统必须先确认选区页关键特征存在，才允许继续点击区服入口

#### Scenario: 选区页未就绪
- **WHEN** 选区页关键特征未出现
- **THEN** 系统必须停止选区流程并报告页面未就绪

### Requirement: 绿色版入口点击
系统 SHALL 支持独立点击绿色版入口。

#### Scenario: 绿色版入口存在
- **WHEN** 系统检测到绿色版入口
- **THEN** 系统必须允许执行点击绿色版入口动作

#### Scenario: 绿色版入口不存在
- **WHEN** 系统未能识别绿色版入口
- **THEN** 系统必须停止后续选区动作并报告入口不可用

### Requirement: 绿色一区选择
系统 SHALL 支持在绿色版入口之后独立选择绿色一区。

#### Scenario: 绿色一区存在
- **WHEN** 系统已进入绿色版选择范围并检测到绿色一区
- **THEN** 系统必须允许执行点击绿色一区动作

#### Scenario: 绿色一区不存在
- **WHEN** 系统未能识别绿色一区
- **THEN** 系统必须停止后续流程并报告目标区服不可用

### Requirement: 选区结果校验
系统 SHALL 在点击绿色一区后校验是否进入正确后续状态。

#### Scenario: 选区成功
- **WHEN** 系统点击绿色一区后出现选中态、页面切换或主页特征
- **THEN** 系统必须判定选区成功并允许进入后续业务流程

#### Scenario: 选区未成功
- **WHEN** 系统点击绿色一区后页面没有发生预期变化
- **THEN** 系统必须判定选区失败并停止后续业务流程

### Requirement: 原子级选区调试模式
系统 SHALL 提供原子级测试模式以支持逐步调试选区流程。

#### Scenario: 单项测试模式
- **WHEN** 开发者运行绿色版入口或绿色一区的单项测试模式
- **THEN** 系统必须只执行该单项动作及其校验，而不执行完整选区流程

#### Scenario: 完整选区前置要求
- **WHEN** 开发者准备运行完整选区流程
- **THEN** 选区页 ready、绿色版入口、绿色一区三个单项测试必须已具备独立可运行能力
