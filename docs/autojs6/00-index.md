# AutoJS6 文档索引

## 目的

这一页不是 API 参考，而是本地查阅地图。

遇到问题时，优先按下面顺序查：

1. 先看本地沉淀文档
2. 再跳官方对应章节
3. 只有本地和官方都不够时，再做额外验证

## 核心结论

- AutoJS6 是 Android 平台的 JavaScript 自动化工具
- 脚本语言是 JavaScript
- 脚本引擎是 Rhino
- 兼容 ES5 全量、ES6 部分特性
- 它既是自动化工具，也是脚本运行环境和轻量 IDE
